Good skills using MATLAB (1)

Posted on 2017-12-19

Intro

This article inclues some useful skills that I would like to use when plotting figures in Matlab, including Legend, Axis, Color of plot, etc.

Legend

Generate Multi-Column in Legend
(1) Download the columnle​gend archive
(2) Unzip, and put the columnlegend.m file into the same fold of your plot
(3) Utilized the columnlenged function to manupilate whatever style for legend

Here is an example
x=1:10;
y1=1:10;
y2=2:2:20;
y3=3:3:30;
y4=4:4:40;
hold on;
f1=plot(x,y1,’-‘,’LineWidth’,2,’Color’,’k’);
f2=plot(x,y2,’-‘,’LineWidth’,2,’Color’,’b’);
f3=plot(x,y3,’-‘,’LineWidth’,2,’Color’,’g’);
f4=plot(x,y4,’-‘,’LineWidth’,2,’Color’,’r’);
name={‘y1’,’y2’,’y3’,’y4’};
Leg=columnlegend(2,name);
set(Leg,’Fontsize’,16,’Fontname’, ‘Cambria’,’FontWeight’,’Bold’);

Axis setting

Here are some setting I would like to use for Axis setting

Example
set(gca,’FontSize’,18,’Fontname’, ‘Cambria’,’FontWeight’,’Bold’);
xlabel(‘xlabel’,’Fontsize’,18,’Fontname’, ‘Cambria’,’FontWeight’,’Bold’);
ylabel(‘ylabel’,’Fontsize’,18,’Fontname’, ‘Cambria’,’FontWeight’,’Bold’);
axis([0 120 3.5 4.3]);
set(gca,’YTick’,[3.5 3.7 3.9 4.1 4.3]);

Plot Color

Besides k b g m r, we can also use RGB value to define a color.

For example, grey can be defined as below
f01=plot(x,y,’-‘,’LineWidth’,2,’Color’,[96 96 96]/255);
