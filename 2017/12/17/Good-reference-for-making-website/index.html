    After reading all these documents
    (1) I download the Mathjax latest version, which is V2.6.
    (2) Then, I unzip it into the fold (C:\Users\laisuo\Blog2\blog\MathJax).
    (3) After that, I go to MarkdownPad2, and click Tools > Options > Advanced > HTML Head Editor. Then, I input the code with the correct site

C:\Users\laisuo\Blog2\blog\MathJax\MathJax-2.6-latest\unpacked\MathJax.js?config=TeX-AMS-MML_HTMLorMML

(4) Then, I can use the Mathjax to edit math forms.
One thing need to mention here that I have to click F6 to see the effect. The live effect of MarkdownPad 2 is not good.
