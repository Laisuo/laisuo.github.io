24-704 Probability and Estimation Methods for Engineering Systems

Posted on 2018-10-10 | In Subject/Course | |

Instructors: Prof. Irving J. Oppenheim

Probability and Random Variables

Basic concept

  • Outcomes: result of experiment
  • Event: a collector of outcomes that are interested to us
  • Sample space \(\Omega\): all possible sample points (all outcomes)

Rules of probability

Axions

  • P{A}: Probability of envent A in one trail is always \(\geq\) 0
  • P{\(\Omega\)} = 1
  • For A and B mutually excluive, P{A \(\bigcup\) B}=P{A}+P{B}

Rules

  • P{\(\phi\)} = 0
  • P{\(\overline{A}\)} = 1-P{A}
  • P{A \(\bigcup\) B} = P{A}+P{B}-P{A \(\bigcap\) B}
  • P{A \(\bigcap\) B}=P{A}\(\cdot\)P{B}, for A and B independent

Combinatorics

Combinations without replacement
- C(n,k)=\((_k^n)\)=\(\frac{n!}{k!(n-k)!}\)

Conditional probability

\[P \lbrace A|B \rbrace=\frac{Number\ of\ outcomes\ in\ A \cup B}{number\ of\ outcomes\ in\ B}=\frac{P \lbrace A \cap B \rbrace}{P \lbrace B \rbrace}\]

Thus, we have \[P \lbrace A \cap B \rbrace =P \lbrace B \rbrace \cdot P \lbrace A|B \rbrace\]

If A and B are independent, we have \[P \lbrace A|B \rbrace=P \lbrace A \rbrace\]

Bayes rule

when we do not have \(P \lbrace A \cap B \rbrace\) \[P \lbrace B|A \rbrace=\frac{P \lbrace A|B \rbrace \cdot P \lbrace B \rbrace}{P \lbrace A \rbrace}\]

Total probability

when we do not have P{A}
Consider mutually exclusive, collects exhaustive events \(B_1,\ B_2,\ B_3,\ ...,\ B_k\) \[P \lbrace A \rbrace=\sum_{j=1}^k [P \lbrace A|B_j \rbrace \cdot P \lbrace B_j \rbrace]\] This gives us the Bayes rule \[P \lbrace B|A \rbrace=\frac{P \lbrace A|B \rbrace \cdot P \lbrace B \rbrace}{\sum_{j=1}^k [P \lbrace A|B_j \rbrace \cdot P \lbrace B_j \rbrace]}\]

Discrete Random Variables and Their Distribution

Discrete Random Variables

Notation
- X: a random variable - x: any particule discrete value of X \[P \lbrace X=x \rbrace=P(x)\]

Probability mass function (pmf)

The probability of x with respect of x

Cumulative distribution function (pdf)

\[F(x)=P \lbrace X \leq x \rbrace =\sum_{y \leq x} P(y) \]

Random Vector

This refers to multiple random variables, like (x,y) \[P \lbrace x,y \rbrace =P \lbrace X=x \cap Y=y \rbrace = P_{x,y}(x,y)\]

  • Marginal distribution P_X(x)= pmf of x for any possible value of y
    P_Y(y)= pmf of y for any possible value of y
    If X, Y are independent, \[P_{X,Y}(x,y)=P_X(x) \cdot P_Y(y)\]

  • Expectation \[E(x)=\mu=\sum_{all\ x} x \cdot P(x)\]

  • Variance
    \[Var(x) = \sigma ^2 = E[(x-E(x))^2] = \sum_{all\ x} (x-\mu)^2 \cdot P(x)\]

  • Standard deviation \[std.(x)=\sigma=\sqrt{Var(x)}\]

  • Covariance \[Cov(X,Y)=E(X \cdot Y)-E(x) \cdot E(Y)\]

  • Correlation coefficient \[\rho=\frac{cov(X,Y)}{std(x) \cdot std(Y)}\]

Chebyshev’s inequality

\[P \lbrace |x-\mu|>\epsilon \rbrace \leq (\frac{\sigma}{\epsilon})^2\] This Chebyshev’s inequality gives you an idea the distribution curve known \(\mu\) and \(\sigma\).

Parameterized discrete random variables (RV)

Bernoulli trial (Bernoulli distribution)

RV X is binary 0, 1 (failure - success, false - true, No - Yes)
Parameter p
- P{x=1}=p - P{x=0}=1-p=q - E{x}=p - Var{x}=p(1-p)=pq

Binomial distribution (Binomial distributed RV)

Perform n independent Bernoulli trials, RV X stands for the number of “success” in the n trials, and X can be 0, 1, 2, …, n. We call this as Binomial (n,p). - Probability mass function (pmf)
\[P(x) = P \lbrace X=x \rbrace = (_x^n) p^x q^{n-x}\] The expectation and variance are
\(E(x)=\mu=np\)
\(Var(x) = \sigma^2 = npq\)

Geometric distribution

P{first success occurs on trial x} - pmf \[P(x)=q^{x-1} \cdot p\] The expectation and variance are
\(E(x)=\mu=\frac{1}{p}\)
\(Var(x) = \sigma ^2 = \frac{1-p}{p^2}\)

Poisson distribution (rare events)

\(\lambda\): average number of arrivals in a fixed time interval
X: number of arrivals in that time interval - pmf \[P(x)=\frac{e^{-\lambda} \lambda^x}{x!}\] E(x)=\(\lambda\) Var(x)=\(\lambda\)

Poisson Approximation to Binomial

Binomial (n,p) = Poisson (\(\lambda\)), where \(\lambda\)=np (for large n and small p).

Continuous Random Variable and Distributions

Continuous RV

  • pdf: probability denstiy function. f(x) \[P \lbrace a<x<b \rbrace = \int_a^b f(x)dx\] Note: (1) f(x) is always positive, (2) the total area under f(x) is 1.0

  • cdf: cumulative distribution function. F(x) \[F(x)=P \lbrace X<x \rbrace = \int_{lower\ limit\ of\ x}^x f(x)dx\]

Uniform distributed RV

  • pdf \[f(x)=a\] where a is a costant

Exponential distributed RV

  • pdf \[f(x)=\lambda e^{-\lambda x}\]
  • cdf \[F(x)=1-e^{-\lambda x}\] where x is the time between each arrival (poisson arrivals).

For exponential distributed RV \(\mu=E(x)=\frac{1}{\lambda}\), this stands for the mean time between arrivals.

Gamma distributed RV

Random variable X: Time of the \(\alpha^{th}\) arrival in a poisson process. \(Gamma(\alpha,\lambda)\)
- pdf \[f(x)=\frac{\lambda^{\alpha}}{\Gamma(\alpha)} \cdot x^{\alpha -1} \cdot e^{-\lambda x}\] where the Gamma function \(\Gamma(n)=(n-1)!\) - \(\alpha\) would be an integer, if we are talking the sum of \(\alpha\) exponentially distributed random variables with parameter \(\lambda\)
- \(\alpha\) need not to be an integer, if we are discribing any random variable that can be modelled by a Gamma shapped pdf, which is a “general purpose” pdf over the range of [0, \(\infty\)]
- Expectation value \[\mu=\frac{\alpha}{\lambda}\] - Variance: \[\sigma ^2=\frac{\alpha}{\lambda^2}\]

Gamma-Poisson Formula

Let T be the RV that described by the Gamma distribution with an integer parameter \(\alpha\) and some positive \(\lambda\). This is a distribution of the time of the \(\alpha^{th}\) rare event. Then, the event {T>t} menas that the \(\alpha^{th}\) rare event occurs after the moment t, and therefore, fewer than \(\alpha\) rare events occur before the time t. \[P \lbrace T>t \rbrace = P \lbrace X<\alpha \rbrace\]

Normal distribution RV

  • pdf \[f(x)=\frac{1}{\sigma \sqrt{2 \pi}} exp{\frac{-(x-\mu)^2}{2 \sigma^2}}\]
  • cdf \[F(x)=\Phi(\frac{x-\mu}{\sigma})\] where \(\Phi(z)\) can be found from Table A4

The expectation value \(E(x)=\mu\) and variance \(Var(x)=\sigma^2\)

Central Limit Theorem

In probability theory, the central limit theorem (CLT) establishes that, in some situations, when independent random variables are added, their properly normalized sum tends toward a normal distribution (informally a “bell curve”) even if the original variables themselves are not normally distributed. \[S_n=\sum_{i=1}^n X_i\] As n becomes very large, the standardized sum coverges into a standard normal random variable. \[Z_n=\frac{S_n-E(S_n)}{std(Sn)}=\frac{S_n-n\mu}{\sigma \sqrt{n}}\] where \(E(S_n)=n\mu\) and \(std(S_n)=\sqrt{n} \cdot \sigma\)

Continuity correction

This correction is needed when we approximate a discrete distribution (like Binomial) by a continuous distribution (Normal). When a continuous distribution (say, Gamma) is approximated by another continuous distribution (Normal), the continuity correction is not needed.

Stochastic procsses

We have leant probability, random variables, random vectors, and their distributions. Are these enough for us to describe a situation involving uncertainty and be able to make good decisions? No. Becasue all what we have leant is the probability/uncertainty at a specific moment. However, many variables develop and change in real time: air temperatures, stock prices, interest rates, football scores, popularity of politicians, and also, the CPU usage, the speed of internet connection, the number of concurrent users, the number of running processes, available memory, and so on. Thus, we need to know stochastic processes, which are random variables that evolv and change in time.

Markov Chain

Stochastic process X(t) is Markov if for any \(t_1\ <\ ...\ <\ t_n\ <\ t\) and any sets A; \(A_1,...,A_n\) \[P \lbrace X(t) \in A|X(t_1)\in A_1,...,X(t_n)\in A_n \rbrace=P\lbrace X(t) \in A | X(t_n) \in A_n \rbrace\] It means that the conditional distribution of X(t) only depends on the latest observation of X.
To be more straitforward, \(P\lbrace future|past,\ present \rbrace = P \lbrace future|present \rbrace\). This property makes a Markov chain a “memoryless” process.

A Markov chain is a discrete-time, discrete-state Markov stochastic process. In our study of this topic, the change in the pmf, from one step to the next, is modeled by a transition probability matrix. For example, the following shows a three-state transition probability matrix.

\[ \left[ \begin{matrix} P_{11} & P_{12} & P_{13} \\ P_{21} & P_{22} & P_{23} \\ P_{31} & P_{32} & P_{33} \end{matrix} \right] \]

  • Know the initial state \(P_0(x)\), the probability matrix for the state after n steps is \[P_n(x)=P_0(x) \cdot [P_{ij}]^n\]
  • Steady state distribution \(\Pi=[\pi_1\; \pi_2\; \pi_3\; ...]\), which is defined as \[P \cdot [\Pi] = \Pi\]

Counting processes

Defination: a stochastic process X is counting if X(t) is the number of items counted by the time t.

  • Binomial process (discrete-time)
    Binomial process X(n) is the number of successes in the first n independent Bernoulli trials, where n=0, 1, 2,…
    It is a discrete-time discrete-space counting stochastic process. Moreover, it is Markov, and therefore, a Markov chain.
    To solve Binomial process, we need some notations, as listed below
  1. \(\Delta\) = frame size.
  2. \(\lambda\) = arrival rate. \(\lambda\)=\(p/Delta\), this is the average number of successes per one unit of time.
  3. \(p\) = probability of arrival (success) during one frame (trial). \(p=\lambda \cdot \Delta\)
  4. \(Y\) = number of frames between successive arrivals
  5. \(T\) = interarrival time
  • Poisson process (continous-time) Going from discrete time to continuous time, Poisson process is the limiting case of a Binomial counting process as frame size \(\Delta\) close to 0.
    Poisson process is a continuous-time counting stochastic process obtained from a Binomial counting process when its frame size \(\Delta\) decreases to 0 while the arrival rate λ remains constant.
